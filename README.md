# 💪 FitTrack — App de seguimiento fitness

App web personal para llevar el seguimiento de entrenamientos, comidas, compras y progresión física. No requiere instalación ni servidor — se abre directo en el navegador.

## ✨ Funcionalidades

- **📅 Calendario** — Marcá cada día con eventos de entreno, comida especial o compras. Con notas por día.
- - **📈 Progresión** — Registrá ejercicios (nombre, tipo, series, reps/peso). Gráfico de evolución de peso corporal. Contador de entrenamientos del mes.
  - - **🍗 Plan de comidas** — Menú semanal de batch cooking (pollo + carne + arroz + verduras). Incluye orden de cocción para el domingo.
    - - **🛒 Lista de compras** — Lista pre-cargada con cantidades exactas para una semana. Se puede tachar lo que ya tenés.

      - 1. Cloná o descargá este repositorio.
        2. 2. Abrí `index.html` en tu navegador.
           3. 3. ¡Listo! Los datos se guardan automáticamente en `localStorage` del navegador.
